# K뷰티 제휴마케팅 쇼츠 3개 제작 패키지

이 폴더는 StyleKorean 신청 전 채널에 K뷰티 콘텐츠 흔적을 만들기 위한 쇼츠 3개 제작 자료입니다.

## 사용 순서

1. `shorts-01`부터 `script.txt`를 읽고 영상 제작
2. `captions.srt`를 자막 파일로 사용
3. `scene-plan.txt`를 보고 컷/이미지 구성
4. `upload-title.txt`, `description.txt`, `hashtags.txt`를 유튜브 업로드에 사용
5. `thumbnail-text.txt`로 썸네일 큰 문구 제작

## 주의

- 재고 구매, 수익 보장, 효능 보장 표현은 쓰지 않습니다.
- 제휴 링크가 아직 없으면 설명란에 "테스트 중"이라고 표시합니다.
- 승인 후에는 Rewards Link를 설명란에 넣고 제휴 고지를 추가합니다.
