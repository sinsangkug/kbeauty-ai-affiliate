# K뷰티 AI 제휴마케팅 사장실

초보자가 K뷰티 제휴마케팅을 시작하기 전에, 재고를 사지 않고 먼저 콘텐츠와 제휴 링크로 시장 반응을 검증하기 위한 단일 HTML 도구입니다.

## 바로 열 파일

`처음_열기.html`

이 파일 하나만 열면 됩니다.

## 무엇을 할 수 있나

| 기능 | 설명 |
|---|---|
| 초보자 마법사 | 다음에 뭘 해야 하는지 단계별로 안내 |
| StyleKorean 조건 확인 | 소셜 채널, 팔로워, 게시물, PayPal 준비 여부 체크 |
| COSRX 콘텐츠 원고 | 블로그 원고와 쇼츠 대본 제공 |
| 제휴 링크 삽입 | 실제 제휴 링크를 넣으면 원고와 성과 입력칸에 반영 |
| 발행 전 검수 | 제휴 고지, 위험 표현, 실제 링크 여부 확인 |
| 성과 추적 | 조회수, 클릭수, 구매수, 커미션을 기록하고 CTR/CVR 계산 |

## 사용 순서

1. `처음_열기.html` 파일을 브라우저로 엽니다.
2. 첫 화면의 “상국님 오늘 딱 이것만 하세요” 박스를 봅니다.
3. 마법사 버튼을 따라갑니다.
4. StyleKorean 가입 조건을 체크합니다.
5. 제휴 링크가 생기면 COSRX 원고에 붙여넣습니다.
6. 콘텐츠를 발행한 뒤 성과 추적 탭에 숫자를 입력합니다.

## 상국님이 직접 해야 하는 일

| 작업 | 이유 |
|---|---|
| StyleKorean 로그인/가입 | 계정 권한 필요 |
| PayPal 이메일 확인 | 정산 정보 |
| 제휴 신청 제출 | 외부 서비스에 정보 전송 |
| 실제 발행 버튼 클릭 | 최종 승인 필요 |

## AI가 도와주는 일

| 작업 | 설명 |
|---|---|
| 원고 작성 | COSRX 블로그/쇼츠 초안 제공 |
| 제휴 고지 확인 | “제휴 링크 포함” 문구 누락 방지 |
| 위험 표현 확인 | 치료, 완치, 즉시 효과 같은 표현 방지 |
| 성과 계산 | CTR, CVR 등 기본 지표 계산 |
| 다음 행동 추천 | 클릭/구매 결과에 따라 수정 또는 확장 판단 |

## 현재 첫 테스트 상품

COSRX Low pH Good Morning Gel Cleanser

처음에는 여러 상품을 동시에 하지 않고, COSRX 클렌저 1개로 콘텐츠와 링크 반응을 확인하는 구조입니다.

## 중요한 주의사항

- 재고를 먼저 사지 않습니다.
- 제휴 링크가 생기기 전에는 콘텐츠를 발행하지 않는 것이 좋습니다.
- 비밀번호, PayPal 인증번호, 개인 인증 정보는 AI에게 공유하지 않습니다.
- 제품 효능을 과장하지 않습니다.
- 치료, 완치, 미백 보장, 즉시 효과 같은 표현은 사용하지 않습니다.
